# QT-
基于http连接百度api的天气预报
主要是通过api来进行查询天气情况。
1先获取本地ip（如果是手机端可以通过GPS定位）来定位，然后通过百度api获取定位信息和天气情况。
返回给界面窗口。
<img width="612" alt="image" src="https://github.com/haoeh/QT-/assets/130277562/74e0a734-ecf3-4700-8bb2-8ce726f2b482">

使用数据都是通过api接口获取的，然后有一个子线程负责实时更新时间

然后提供了一个搜索当地天气的功能
